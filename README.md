# react-products

Simple [React] app to list products

Requirements:
- git
- npm v3

To run the app locally:

```bash
git clone https://github.com/fishyFrogFace/product.git
cd products
npm install
npm start
```

The vector images can be found on [Pixabay] and are released under [Creative Commons CC0]
